# Data Directory

## Structure

```
data/
├── input_videos/     # Place input videos here
├── output_videos/    # Processed videos saved here
└── sample_frames/    # Sample frames for testing
```

## Sample Video

Download sample video from:
- [Roboflow](https://universe.roboflow.com/roboflow-jvuqo/football-players-detection-3zvbc)
- [SoccerNet](https://www.soccer-net.org/)

## Supported Formats

- MP4, AVI, MOV, MKV
- Recommended: 1080p or 720p for best performance
